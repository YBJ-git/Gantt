## 🏗️ 기술 아키텍처

### 프론트엔드 아키텍처
```
src/
├── components/          # 재사용 가능한 컴포넌트
│   ├── common/         # 공통 컴포넌트
│   ├── layout/         # 레이아웃 컴포넌트
│   ├── modals/         # 모달 컴포넌트
│   ├── realtime/       # 실시간 기능 컴포넌트
│   ├── pwa/           # PWA 관련 컴포넌트
│   ├── feedback/      # 피드백 시스템 컴포넌트
│   └── optimized/     # 성능 최적화된 컴포넌트
├── contexts/           # React Context (전역 상태)
│   ├── AuthContext.js
│   ├── RoleContext.js
│   ├── NotificationContext.js
│   └── RealtimeContext.js
├── services/           # API 및 외부 서비스
│   ├── authService.js
│   ├── userService.js
│   ├── taskService.js
│   ├── feedbackService.js
│   └── websocket/
├── pages/              # 페이지 컴포넌트
│   ├── Auth/          # 인증 관련 페이지
│   ├── Profile/       # 프로필 관련 페이지
│   └── Dashboard/     # 대시보드 페이지
└── tests/              # 테스트 파일
    ├── api/           # API 테스트
    └── components/    # 컴포넌트 테스트
```

### 백엔드 아키텍처
```
src/
├── api/               # API 엔드포인트
├── controllers/       # 컨트롤러 로직
├── services/          # 비즈니스 로직
├── repositories/      # 데이터 액세스 레이어
├── middleware/        # 미들웨어
├── websocket/         # WebSocket 서버
├── config/            # 설정 파일
├── utils/             # 유틸리티 함수
└── tests/             # 백엔드 테스트
```
